TROUBLESHOOTING QUICK REFERENCE
================================================================================

Authentication fails:
→ Check LOGIN_EMAIL and LOGIN_PASSWORD in .env
→ Try logging in manually at pendulumedu.com
→ Verify account is active

Font errors:
→ Install Noto Sans Gujarati font
→ Restart terminal after installation
→ Check font in Control Panel → Fonts

Telegram "Unauthorized":
→ Verify TELEGRAM_BOT_TOKEN is complete
→ Check for extra spaces in .env
→ Ensure token format: 123456789:ABCdefGHIjklMNOpqrsTUVwxyz

Telegram "Chat not found":
→ Verify channel username in .env
→ Ensure bot is added to channel
→ Check bot has admin permissions
→ Verify bot can post messages

Module not found:
→ Activate virtual environment: venv\Scripts\activate
→ Install dependencies: pip install -r requirements.txt

No quizzes found:
→ Check authentication succeeded
→ Verify website is accessible
→ Check if website structure changed
